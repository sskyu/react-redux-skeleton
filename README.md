# react-redux-skeleton

## Start
```sh
# Install yarn
$ npm i -g yarn

# Install dependencies
$ yarn
...

# Building assets and running server
$ yarn start
...

# Open http://localhost:8080 in your browser
```

* [x] React v16
* [ ] Redux
* [x] Babel v7
* [x] Webpacker v3
